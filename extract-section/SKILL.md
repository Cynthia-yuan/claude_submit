# 章节提取 Skill

## 功能说明

从 Word 文档（.docx）中提取指定章节的内容：
1. 自动识别章节编号（如 5.1.2.1）
2. 提取该章节范围内的所有表格
3. 按变更类型分类：删除、新增、变更
4. 输出易读的分类摘要

## 执行步骤

1. **确认文档路径**
   - 检查指定的 .docx 文件是否存在
   - 验证文件格式是否正确

2. **查找章节**
   - 在文档中搜索指定的章节编号
   - 确定章节的起始和结束位置
   - 提取章节标题

3. **提取表格**
   - 识别章节范围内的所有表格
   - 解析表格的表头和数据行
   - 过滤掉表格分隔行

4. **分类数据**
   - 查找"类型"或"变更类型"列
   - 根据关键词自动分类：
     - 删除：包含"删除"、"移除"、"废弃"
     - 新增：包含"新增"、"添加"、"引入"
     - 变更：包含"修改"、"变更"、"更新"

5. **输出结果**
   - 显示分类摘要（各类项数量）
   - 列出每项的详细信息
   - 按类型分组展示

## 注意事项

- 文档必须是 .docx 格式（Word 2007+）
- 章节标题必须以编号开头（如 "5.1.2.1 标题"）
- 表格需要有"类型"或"变更类型"列才能自动分类
- 只提取指定章节范围内的表格，不包含子章节

- "提取章节"
